Indian Language Icons

Illustrations

A set of illustrated icons denoting some of the Indian Languages. Each icon is made from the first alphabet fof the language name. A small side project done for OLA Play. These illustrations are used as album artworks for regional playlists by OLA Play.

"Hi"-Hindi

"Ka"-Kannada

"Pa" - Punjabi

"Ta"- Telugu

"Ma"-Marathi

"Ma"-Malayalam

"Ta"-Tamil

"Ba"-Bengali
